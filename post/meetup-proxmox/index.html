Next meetup on Campus Madrid: Proxmox VE

26 Nov 2017, 22:00
Virtualization / MeetUps

Hello everyone

Next friday, 1st of December I’ll be at Campus Madrid giving a brief introduction to Proxmox VE and how to use it.

Proxmox is an open-source server virtualization enviroment. It allows the deployment and management of containers (LXC) and virtual machines KVM. Proxmox includes a web console some console and command-line tools and a REST API for third party tools.
